### 2. Claude Code 源码泄露事件持续发酵，泄露细节曝光
**来源**: [Alex Kim's Blog](https://alex000kim.com/posts/2026-03-31-claude-code-source-leak/) | **标签**: Claude, Agent, 安全
Anthropic 的 Claude Code CLI 工具因 npm 包中意外附带 .map 文件，泄露了 51.2 万行源码。安全研究人员发现多项有趣细节：内置反蒸馏机制（anti_distillation），可向 API 请求注入伪造工具定义以污染竞争对手的训练数据；"Undercover Mode" 允许 AI 隐藏其 AI 身份；每天约浪费 25 万次 API 调用；还有名为 KAIROS 的未发布自主 Agent 模式。Anthropic 已撤回该包，但代码已被广泛镜像。这是 Anthropic 一周内第二次意外泄露（此前有 Model Spec 泄露）。

韩国游戏公司 Krafton（PUBG 开发商）推出 AI 模型品牌 "Raon"（韩语"喜悦"之意），并在 Hugging Face 上开源 4 款模型：Raon-Speech（9B 参数语音 LLM，在 10B 以下规模中英韩三语全球第一）、Raon-SpeechChat（首款韩国本土全双工实时语音对话模型）、Raon-OpenTTS（纯开源数据集训练的 TTS 模型）和 Raon-Vision Encoder（视觉编码器）。Raon-Speech 在 7 项核心任务、40 个基准测试的平均排名中位列 10B 以下模型全球榜首。

### 4. Meta 开源 BOxCrete：AI 优化美国本土水泥配方，减少进口依赖
**来源**: [Meta Engineering](https://engineering.fb.com/2026/03/30/data-center-engineering/ai-for-american-produced-cement-and-concrete/) | **标签**: Meta, 开源, 行业应用
Meta 在 2026 年美国混凝土协会春季大会上发布了 BOxCrete（Bayesian Optimization for Concrete）——一款用于优化混凝土配方设计的贝叶斯优化 AI 模型，并同步开源基础数据。美国每年浇筑约 4 亿立方米混凝土，但约 20-25% 的水泥依赖进口。BOxCrete 可帮助混凝土供应商快速探索和验证新配方，减少对进口水泥的依赖，同时满足强度、成本和可持续性等多重约束。美国水泥行业年产值超 1300 亿美元，相关就业岗位约 60 万个。

### 6. KV Cache 优化前沿：从 300KB 到 69KB/Token 的演进
**来源**: [Future Shock](https://news.future-shock.ai/the-weight-of-remembering/) | **标签**: LLM, 底层架构, 技术深度
一篇技术深度文章揭示了 LLM 中 KV Cache（键值缓存）的物理重量问题：GPT-2 每 Token 需要 300 KiB 的 GPU 显存来存储缓存，4000 Token 对话仅缓存就占用 1.2GB 显存。演进路线为：GPT-2（300 KiB/token）→ Llama 3 GQA（128 KiB/token）→ 持续优化。文章指出，每次缓存的字节都对应着真实的电费和 GPU 成本消耗，这也是各厂商竞相优化 KV Cache 架构的核心动力。
